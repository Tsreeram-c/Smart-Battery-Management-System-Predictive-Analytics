# Smart Battery Management System with Predictive Analytics

## Overview

This project presents a Smart Battery Management System (BMS) developed for a 6S Li-ion battery pack using ESP32, INA226 current/voltage sensor, and DS18B20 temperature sensor.

The system performs real-time monitoring of battery parameters including voltage, current, power, and temperature while providing predictive battery analytics using Python-based data analysis and machine learning techniques.

## Key Features

* Real-time battery voltage monitoring
* Real-time current and power measurement
* Temperature monitoring using DS18B20
* OLED display for live battery status
* Wi-Fi based remote monitoring
* Historical battery data logging
* Battery data visualization using Python
* Anomaly detection for abnormal battery behavior
* Predictive analytics for battery temperature trends
* Battery health monitoring concepts (SOC, SOH, RUL)
